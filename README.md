# Fake Job Detection using NLP & Machine Learning

## Overview
This project detects fraudulent job postings using NLP and Logistic Regression.

## Workflow
1. Data Cleaning
2. EDA
3. TF-IDF Vectorization
4. Model Training
5. Model Evaluation
6. Deployment using Streamlit

## Accuracy
~90%

## Technologies
Python, Scikit-learn, NLP, Streamlit
